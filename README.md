# Rope Platformer

2.5D платформер-песочница на Unity, в котором игрок проходит уровни с помощью физического тулгана. Основная идея игры: связывать объекты веревками и пружинами, менять длину связей и использовать получившиеся конструкции для решения задач.

Выполнил: Кудинов Д.В.

Учебная группа: М8О-412Б-22

## Описание

Игрок управляет персонажем сбоку и взаимодействует с физическими объектами уровня. Связи между объектами имеют нагрузку, могут растягиваться, раскачиваться и разрушаться при перегрузке. В игре есть обучающий уровень и второй уровень с переносом грузового ядра через несколько испытаний.

## Основные механики

- создание физических связей между объектами;
- два типа связей: веревка и пружина;
- изменение длины выбранных связей;
- выбор и настройка нескольких связей одновременно;
- визуальная индикация нагрузки;
- разрушение перегруженных связей;
- доставка грузового объекта через уровень;
- разводной мост и пружинная пусковая платформа.

## Управление

- `A` / `D` или стрелки - движение;
- `Space` / `W` / стрелка вверх - прыжок;
- `1` - выбрать веревку;
- `2` - выбрать пружину;
- левая кнопка мыши - выбрать объект или создать связь;
- `Shift + левая кнопка мыши` - выбрать несколько связей;
- `Q` - укоротить выбранные связи;
- `E` - удлинить выбранные связи;
- колесо мыши - изменить длину выбранной связи;
- средняя кнопка мыши - удалить связь под курсором;
- `Delete` / `Backspace` - удалить выбранные связи;
- `C` - удалить все связи;
- `Tab` - посмотреть выше камерой.

## Сцены

- `MainMenu` - главное меню;
- `MainScene` - обучающий уровень;
- `Level2Scene` - основной уровень с грузовым ядром, разводным мостом и пружинной платформой.
